# NPL CAD

NPL CAD 是[paracraft](http://paracraft.cn/)的Mod插件。 目的是让用户可以
- 通过编程的方式进行CAD建模
- 学习计算机编程语言
- CAD模型可3D打印，或导入到Paracraft世界中变成bmax模型
